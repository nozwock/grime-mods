# GRIME Mods

[BepInEx] plugins for [GRIME][steam-grime], a Unity game.

## Prerequisite

GRIME doesn't seem to work with BepInEx out of the box, so you'll need this build of BepInEx that includes a preloader
patch to make it work with the game.
- https://github.com/BadMagic100/Grime.ModdingTools


[BepInEx]: https://github.com/BepInEx/BepInEx
[steam-grime]: https://store.steampowered.com/app/1123050/GRIME/
